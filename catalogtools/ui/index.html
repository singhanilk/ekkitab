Internal Utils(Prasad Potula - 05/Oct/2010 )

    Set up and source code for these tools

    1. The internalutils directory is a soft link to the /home/ubuntu/prasad/src/scm/catalogtools/ui directory.
    The reason being this source code is used for uploading catalogs, image and new arrivals stock data.

    Dependencies and Assumptions

    1. A development database has been created which is a replica of refernce database. All the scripts which
    modify database point to the development database
    2. Scripts like copy images etc work on the main /mnt directories only
    3. A script called setenv.php is included in all the phtml scripts which sets the environment to development.

    List of Utilities

    1. isbncheck.php - Isbn Check Program for checking a particular isbn in reference database
    2. missingimages.phtml - Program which runs google get images for all missing images for a selected supplier
    3. pdftoimages.phtml - Program takes a PDF file and extracts images, after specifying the respective isbn can be uploaded
    4. bookupdate.phtml - Data entry for a single book
